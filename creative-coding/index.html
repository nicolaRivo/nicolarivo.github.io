ABOUT CREATIVE CODING

These experiments represent my exploration of the creative possibilities at the intersection of code, sound, and visual art. Using technologies like p5.js and the Web Audio API, I create interactive experiences that respond to user input and generate unique audio-visual outcomes.

Each piece in this collection is an opportunity to experiment with different coding techniques and algorithmic approaches to creative expression. Many of these experiments directly inform my sound design practice, allowing me to prototype new ideas and discover unexpected sonic possibilities.

All sketches are built with open web technologies and are interactive - you're encouraged to play, modify, and explore them. The source code for most experiments is also available on GitHub.

Technologies Used:
